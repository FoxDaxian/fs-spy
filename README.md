# fs-spy

[![NPM version](https://img.shields.io/npm/v/fs-spy?color=a1b858&label=)](https://www.npmjs.com/package/fs-spy)

Monitoring fs accessing for Node process

## Install

```bash
npm i -D fs-spy
```

## Usage

```bash
node -r fs-spy my-module.js
```
